## Overview

EduMap is a functional mobile app developed for **Technovation Girls** to reconnect younger generations with their local cultural heritage through interactive storytelling.

## Tech Stack

* **Platform:** Developed using **MIT App Inventor** (block-based logic).
* **Features:** Includes a notification system for landmarks, interactive UI for user content, and geolocation-based data mapping.

## Impact

* **Education:** Makes learning about local traditions fun and engaging.
* **Wellness:** Encourages a healthy lifestyle through physical exploration of historical sites.
* **Access:** Bridges the gap to the past via universal smartphone accessibility.

## Recognition
